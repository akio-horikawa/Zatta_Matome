# GitHub連携
```
// GitHub上でCode→Clone→HTTPSのタブのリンクをコピー
// Vscodeでリポジトリソースをペースト
// 宛先のローカルフォルダを選択してクローンを実行

cd 宛先ローカルフォルダ // 1
git clone URL // 2
// 1,2の順で行えばおｋ

// アカウント情報を登録しておくと草が生やせる
// 以下をVscodeのコンソールに
git config --global user.name "hoge hoge"
git config --global user.email "hoge@hogemail.com"
// ""無しで打てば現在の設定を確認可能
```
